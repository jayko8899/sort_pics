Python project that will organize pictures by date.

TO RUN:

2 required positional arguments: source folder and destination folder

Source folder is where the program will look for unsorted media files

Destination folder is where the program will store media files once they are sorted

1 optional flag: -d

If -d flag set then program will delete all media files in the source folder once it is done copying them to the destination folder 

Example: "python3 media_sort.py ./photos ./sorted_photos"     #unsorted files are in ./photos and will be sorted into ./sorted_photos
